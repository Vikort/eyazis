Система формирования словаря естественного языка

Система позволяет сформировать словарь из текста русского языка, загурзить словарь из файла, а также сохранить словарь в файл.

Словарь содержит список слов, упорядоченный по алфавиту и включающий только лексемы с дополнительно оформленными записями о месте и роли данного слова в составе предложения.
К такой информации относится описание того, каким членом предложения может быть данное слово и в какой форме.

Для составления словаря необходимо ввести текст в верхнее поле и затем нажать кнопку "Преобразовать".
Для сохранение словаря необходимо нажать кнопку "Сохранить", в появившемся окне выбрать нужный файл или задать имя новому файлу.
Для открытия словаря необходимо нажать пункт меню "Файл", в появившемся окне выбрать нужный файл.

Столбец "Лексема" содержит список лексем содержащихся в словаре.
Столбец "Тэги" содержит морфологические признаки лексемы в данном предложении.
Столбец "Член предложения" содержит информацию каким членом предложения в данном предложении лексема является.
